Phase 3 — Game Layer

GyateNFT metadata is intentionally game-ready. Rarity maps to character tier, variant_multiplier maps to stat bonuses, and each NFT has a unique sequential ID.

  • Turn-based or idle game using GyateNFTs as characters
  • Rarity & variant determine base stats & abilities
  • Seasonal event NFTs unlock limited game content
  • Potential $GYATE token for staking & governance
